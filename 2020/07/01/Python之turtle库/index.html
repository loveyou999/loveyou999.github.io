Python之turtle库

用import引入库
turtle（绘图）库的使用

import turtle
turtle.setup(650,350,200,200)
turtle.penup()
turtle.fd(-250)
turtle.pendown()
turtle.pensize(25)
turtle.pencolor("magenta")
turtle.seth(-40)
for i in range(4):
    turtle.circle(40,80)
    turtle.circle(-40,80)
turtle.circle(40,80/2)
turtle.fd(40)
turtle.circle(16,100)
turtle.fd(40 * 2/3)
turtle.done()

from <库名> import <函数名>或者*；

也可以import<库名>和as<库别名>,就可以a.b使用；

以上代码中setup（a,b,c,d）
a、b为宽度和高度,c和d为起始位置坐标，相对于左上角而言。goto（a,b）去这个位置;

penup()抬起笔，pendown()放下笔;

pensize(25)画笔的尺寸大小;

pencolor(“magenta”)画笔的颜色;

fd像正前方向运动，bk是反方向运动，circle（r,angle）是以左侧某一点，r为半径 ，angle为角度进行曲线运动;

seth(-40)改变行进方向，绝对角度;
for i in range（N）
循环N次
range（M,N）
产生M到N-1个整数序列，共N-M个

turtle.done()不关闭窗口，需要自己关闭。
